# 🎓 Sistema de Gestión Académica (C++)
> **Proyecto de Lógica de Programación y Algoritmos - Universidad Técnica de Ambato**

Este software permite el procesamiento automatizado de calificaciones y asistencia para estudiantes universitarios, generando reportes estadísticos detallados por paralelos.

---

## 🚀 Características Principales
* **Cálculo Ponderado:** Implementación de la métrica 30% Parcial 1, 30% Parcial 2 y 40% Prácticas.
* **Validación Robusta:** Uso de `try-catch` y saneamiento de buffer para entradas de datos seguras.
* **Gestión de Memoria:** Implementación de arreglos dinámicos en el *Heap* mediante `new` y `delete[]`.
* **Analítica Académica:** Clasificación por categorías (Excelente, Muy Bueno, etc.) y detección de alertas por asistencia.

## 📊 Estructura del Reporte
El sistema genera un resumen estadístico automático que incluye:
1. **Conteo por Paralelo:** Distribución de estudiantes en las secciones A, B y C.
2. **Promedios:** Cálculo general y específico por cada paralelo.
3. **Métricas de Rendimiento:** Porcentajes de aprobación y estabilidad académica.
4. **Detección de Extremos:** Identificación del mejor y peor promedio del curso.

## 🛠️ Requisitos e Instalación
1. Tener instalado un compilador compatible con **C++11** (Recomendado: **Code::Blocks 20.03**).
2. Clonar el repositorio o descargar el archivo `.cpp`.
3. Compilar y ejecutar para iniciar el procesamiento de datos.
